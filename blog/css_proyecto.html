Proyecto: Galería de Imágenes con HTML + CSS

Publicado el 23 de agosto de 2025 • Por @angelajasso

💡Segundo proyecto completado del roadmap

Después de avanzar en mi aprendizaje de desarrollo web, quise poner en práctica lo que aprendí sobre HTML y CSS creando un proyecto. 🚀

📸 Este proyecto consistió en crear una galería de imágenes responsive, aplicando conceptos de maquetación y diseño que he estado aprendiendo.

🎯 Objetivos principales

  • ✅ Organizar el HTML con etiquetas semánticas
  • ✅ Trabajar con Flexbox y Grid para lograr un layout adaptable
  • ✅ Aplicar container queries para adaptarlo a diferentes dispositivos
  • ✅ Mantener consistencia visual con variables CSS

🚀 Aprendizajes

Aunque es un proyecto sencillo, es un paso clave en mi objetivo de convertirme en Full Stack Blockchain Developer. Documentar mi progreso me motiva y espero que también inspire a otros que estén aprendiendo.
