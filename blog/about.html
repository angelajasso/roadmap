🛤️ Roadmap de Aprendizaje

Mi camino en Desarrollo Web y Blockchain no es solo una lista de tecnologías, sino una historia de crecimiento, retos y aprendizajes compartidos.

🚀 Punto de partida

Antes de escribir una sola línea de código, decidí enfocarme en lo esencial: **entender cómo funciona Internet, la web y la programación desde sus bases**. He trabajado con varios lenguajes (Ensamblador, C, Java, JavaScript, Python), pero al entrar al mundo **Web3 y los contratos inteligentes en Solidity**, me di cuenta de que sin bases sólidas es difícil dar el salto.

💻 Desarrollo Web

Actualmente estoy fortaleciendo mis habilidades en Frontend y Backend:

  • Fundamentos de Internet y la Web.
  • HTML y CSS → estructura y estilos
  • JavaScript → interactividad real
  • Git y GitHub → control de versiones y colaboración
  • React y TailwindCSS → construcción de interfaces modernas
  • Python y Node.js → lógica backend

📌 Cada tema lo aprendo construyendo un proyecto y documentándolo en GitHub.

🌐 Blockchain (Web3)

¿Por qué Blockchain? Muchos piensan que es moda, pero yo creo que es un cambio inevitable:

  • Propiedad digital .
  • Contratos sin intermediarios
  • DAOs y nuevas formas de organización

👉 Mi visión: en pocos años, usar aplicaciones en blockchain será tan normal como abrir cualquier sitio web. 📌 En esta etapa aprenderé:

  • Solidity (contratos inteligentes)
  • Ethereum y Polygon
  • Web3.js / Ethers.js
  • Integración de dApps con Frontend

📚 Filosofía de aprendizaje

No busco aprender rápido, busco aprender bien. Por eso, mi roadmap está dividido en bases → proyectos → comunidad.

  • Bases: teoría y fundamentos.
  • Proyectos: aplicar lo aprendido en código real.
  • Comunidad: compartir en redes sociales, aprender de otros y recibir feedback.
