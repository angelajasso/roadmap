Internet

Publicado el 18 de agosto de 2025 • Por @angelajasso

🧠 ¿Qué pasa cada vez que abres una página web?

Cuando abres tu navegador y escribes una dirección como www.ejemplo.com, sucede mucho más de lo que parece detrás de pantalla. Entender este proceso es fundamental si estás empezando en el mundo del desarrollo web. En este artículo te explico paso a paso cómo funciona Internet desde cero, para que construyas con bases sólidas tus proyectos. 🚀

🌐 ¿Qué es Internet?

Internet es una red global de computadoras conectadas entre sí que se comunican gracias a un conjunto de reglas llamadas protocolos. Podemos imaginarlo como una enorme telaraña de conexiones que permite que la información viaje de un lugar a otro en segundos.

🔄 El viaje de un sitio web

Cuando escribes una URL como: www.ejemplo.com empieza un viaje de datos que involucra diferentes actores y tecnologías.

🖥️ El cliente: tu navegador

El primer protagonista es el cliente, que normalmente es tu navegador (Chrome, Firefox, Safari, etc.).

  • Su tarea es pedir información al servidor donde está alojada la página.
  • El servidor responde enviando los archivos necesarios.

🧭 El papel del DNS

Pero antes de que eso ocurra, necesitamos algo importante: saber a qué servidor dirigirnos. Aquí entra el DNS (Domain Name System), que funciona como una guía telefónica de Internet.

  • Traduce nombres como www.ejemplo.com en una dirección IP (por ejemplo: 192.168.0.1).
  • Con esa IP, el navegador sabe exactamente dónde buscar la página.

📡 Protocolos HTTP y HTTPS

Una vez que el navegador encuentra el servidor, se comunica usando protocolos:

  • HTTP (HyperText Transfer Protocol) → protocolo para transferir datos.
  • HTTPS → la versión segura que protege la información con cifrado. Esto garantiza que lo que ves en tu pantalla no ha sido alterado en el camino.

🔐 Criptografía: la seguridad detrás

La criptografía hace posible que tu información (como contraseñas o datos bancarios) viaje de forma segura en Internet. Gracias a ella, los datos están cifrados y no pueden ser leídos por terceros.

📥 ¿Cómo llega la web a ti?

Finalmente, el servidor envía los archivos:

  • HTML → estructura
  • CSS → estilos
  • JavaScript → interactividad

El navegador interpreta esos archivos y construye la página que ves en pantalla.

🙌 Conclusión

La próxima vez que abras un sitio web, recuerda que detrás de un simple click ocurre un viaje complejo de información. Y si quieres aprender desarrollo web o dar el salto hacia tecnologías como Blockchain, entender estos fundamentos es esencial.
