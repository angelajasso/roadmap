# 🎵 Galería de Imágenes de Instrumentos Musicales

Una galería visual y responsive creada con **HTML5** y **CSS3**, diseñada para mostrar diferentes instrumentos musicales con un diseño limpio, elegante y adaptable a cualquier dispositivo.

---

## 🧱 Tecnologías
- **HTML5** → Estructura semántica y accesible.
- **CSS3** → Diseño responsivo con *Grid Layout* y efectos *hover*.
- **Imágenes optimizadas** para carga rápida.

---

## 🎯 Características
- Diseño **totalmente responsive**.
- **Grid dinámico** que se adapta al ancho de pantalla.
- Uso de **etiquetas semánticas** para accesibilidad.
